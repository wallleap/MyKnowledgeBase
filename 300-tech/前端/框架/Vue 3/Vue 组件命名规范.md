大驼峰（`StarRate`）使用的时候可以

```vue
<StarRate />
<star-rate />
```

短横线（`star-rate`）使用的时候只可以

```vue
<star-rate />
```

## 最佳实践

1. 使用大驼峰命名
2. 使用的时候都可以，但要保持统一（尽量使用大驼峰，可以和普通 HTML 元素区分开；使用短横线需要加前缀，否则容易和 HTML 元素冲突）
3. 自闭合或标签闭合都需要保持统一
4. 模版中只能使用短横线
