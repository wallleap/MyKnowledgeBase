使用 oh-my-env（推荐）

- 基于最新版 Docker
- 已安装 node18、pnpm7、npm8、zsh
- <https://github.com/FrankFang/oh-my-env-1>
- 进入上述仓库查看步骤
- 使用
	- `uname -a` linux 查看用户  `pwd` 当前目录  `whoami` 当前用户
	- `f` 查找文件
	- `fd` 查找目录

前端自己安装（安装最新版本）

- node18+
- bash
- 在 VSCode 里配置终端为 bash
