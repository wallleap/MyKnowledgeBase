| 功能点                | XHR   | Fetch   |
| ------------------ | ----- | ------- |
| 基本的请求能力            | ✅     | ✅       |
| 基本的获取响应能力          | ✅     | ✅       |
| 监控请求进度             | ✅     | ❌       |
| 监控响应进度             | ✅     | ✅       |
| Service Worker 中可用 | ❌     | ✅       |
| 控制 Cookie 的携带      | ❌     | ✅       |
| 控制重定向              | ❌     | ✅       |
| 取消请求               | ✅     | ✅       |
| 自定义 referrer       | ❌     | ✅       |
| 流                  | ❌     | ✅       |
| API 风格             | Event | Promise |
| 活跃度                | 停止更新  | 不断更新    |
