<https://ollama.com/> 下载对应安装包

安装完成后在终端运行对应模型，例如

```sh
$ ollama run deepseek-r1:1.5b
```

可以直接在终端提问

运行命令可以查看所有的模型

```sh
$ ollama list
```

直接在终端提问不方便

- 可以 <https://chatboxai.app/zh> 下载 ChatBox 最新版，并在设置中设置模型为 Ollama Deepseek-R1
- Cherry Studio
- VSCode 则可以下载 Continue 插件
